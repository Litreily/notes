## Install Packages

### 安装路径

``` bash
cd ~/.atom/packages/
```

### 安装方法

``` bash
git clone https://github.com/<user-name>/<package-name>.git
npm install <package-name>
```

### 启用方法

快捷键`Ctrl+Shift+P`打开指令窗口，输入`Window:reload`或者直接用快捷键`Ctrl+Shift+F5`重启`Atom`生效。
