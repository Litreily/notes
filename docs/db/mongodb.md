## mongoexport

``` sh
mongoexport -h localhost:27017 -d databaseName -c collectionName -f data.json
```

## mongoimport

``` sh
$ mongoimport -h localhost:27017 -d databaseName -c collectionName -f data.json
$ mongo
> use databaseName
> databaseName.collectionName.find().pretty()
```
